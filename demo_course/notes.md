# Physics II Release Demo

Fieldnotes keeps indexing, retrieval, notebook artifacts, quizzes, and source reopening tied to exact workspace passages. This demo course simulates five weeks of a pendulum unit with lecture notes, lab data, review sheets, and problem sets.

Week 1 framing:

- guiding question: what changes period and what changes damping
- baseline assumption: small-angle pendulum period is mostly set by length
- warning: amplitude decay and oscillation period are different measurements

Pendulum investigation:

- Trials 1 to 3 show gradual amplitude loss under similar conditions
- Trial 4 damps much faster than Trials 1 to 3 because felt pad friction was added near pivot
- Compare chart output against course notes before claiming resonance or sensor failure

Study workflow:

- Use quiz flow to review damping concept after reading summary PDF
- Reopen source cards when answer cites week 4 slides or problem set
- Check dataset columns before writing analysis about outliers
